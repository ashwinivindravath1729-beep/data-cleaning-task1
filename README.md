# Data Cleaning & Structural Validation - Task 1

## Project Overview
Transform raw, unstructured data into a reliable format for analysis.

## Problems in Raw Data
- Duplicate entries
- Missing values
- Inconsistent formats (dates, gender, phone, email)

## Steps Performed
1. Loaded raw dataset (15 rows, 9 columns)
2. Removed 4 duplicate rows
3. Filled missing values (age, purchase_amount, name, city)
4. Standardized all columns (names, emails, dates, phone numbers)
5. Saved clean output to cleaned_data.csv

## Files
- data.csv → Raw messy dataset
- clean.py → Data cleaning script
- cleaned_data.csv → Final clean output

## Tools Used
- Python
- Pandas
- Dateutil
